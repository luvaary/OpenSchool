  6. ANIMATION & TRANSITION INVENTORY
------------------------------------------------------------

  Design Tokens (css/tokens.css)
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  --transition-fast    : 150ms ease
  --transition-base    : 250ms ease
  --transition-smooth  : 300ms cubic-bezier(0.4, 0, 0.2, 1)
  --transition-spring  : 400ms cubic-bezier(0.34, 1.56, 0.64, 1)

  Keyframes (css/styles.css)
  ~~~~~~~~~~~~~~~~~~~~~~~~~~
  @keyframes fadeIn       : opacity 0 --> 1
  @keyframes slideUp      : translateY(12px) + opacity 0 --> 0 + 1
  @keyframes fadeSlideUp  : translateY(8px) + opacity 0 --> 0 + 1
  @keyframes pulseRing    : scale(1) opacity 0.4 --> scale(1.4) opacity 0

  Utility Classes (css/styles.css)
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  .animate-fade-in       : applies fadeIn 400ms ease-out forwards
  .animate-slide-up      : applies slideUp 500ms ease-out forwards
  .animate-stagger       : container for stagger children
  .animate-child         : child with fadeSlideUp 400ms ease-out forwards
  .animate-child:nth-child(1..8) : stagger delay 60ms increments

  Micro-interactions (css/components/)
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  button  : :active scale(0.97)
  card    : hover translateY(-4px) shadow lift, :active translateY(-1px)
  nav     : hover padding-left +4px
  modal   : spring-eased scale(0.97)+opacity entrance
  form    : smooth focus transition
  table   : row hover background transition
